# eslint-config-react

## Install

1. Install Eslint on your projet

```shell
yarn add --dev eslint @eslint/js
```

2. Install config

```shell
yarn add --dev @nodstudio/eslint-config-react
```

3. Add config to yours

```js
import nodStudioConfigReact from "@nodstudio/eslint-config-react";

export default [
    // ...
    ...nodStudioConfigReact
];
```
